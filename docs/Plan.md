# 🧠 AI Agent Digital Portfolio – Insurance (Frontend)

## 📌 Overview

This project is a **Digital Portfolio for an Insurance Agent**, designed to showcase services, plans, and contact information in a clean, modern, and responsive interface.

The focus of this project is **frontend development only**, using modern web technologies to create a professional and user-friendly experience for potential clients.

---

## 🚀 Tech Stack

* **Next.js** – React framework for building fast, scalable web apps
* **Tailwind CSS** – Utility-first CSS framework for styling
* **Lucide Icons** – Clean and customizable icon library

---

## 🎯 Project Goal

To build a **high-converting digital portfolio** that:

* Presents insurance plans clearly
* Builds trust with potential clients
* Makes it easy to contact the agent
* Works seamlessly across all devices

## 🛠️ Setup Instructions

```bash
# Create project
npx create-next-app@latest insurance-portfolio

# Navigate
cd insurance-portfolio

# Install dependencies
npm install

# Install Tailwind CSS
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

# Install Lucide Icons
npm install lucide-react

# Run development server
npm run dev
```
